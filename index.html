These are a few of my projects!

Cheap Eats

This is a food deal finding application powered by Groupon meant for finding Groupon deals on restaurants close to a given address and generate maps from the given address to those restaurants. It's a simple HTML/CSS and Javascript application attempting to make use of object-oriented programming

Chess AI

A chess AI built in Javascript to be run in the browser. It uses alpha-beta pruning to predict how the player will react and uses board-state caching to play more efficiently. It uses chessboard.js for the UI and chess.js for the game logic.

An Exploration in Java

A tutorial exploring some of the benefits and drawbacks of Java as well as an explanation for how to set up a CRUD API connected with a PostgreSQL database using Java's Spring web server framework and JDBC.

Trofi Bouffe

A recipe building application that allows user to make, save, and edit recipes, as well as make calls to the Nutritionix API to set aggregate nutrition facts for chosen Recipes. React.js is used on the front-end and Node.js and PostgreSQL are used for the backend.

Other Experiences, Activities, and Hobbies

Church Plant Intern at Acts2Fellowship Santa Barbara

Also cuisine coordinator and audio technician

In my time between graduating from college and studying web development, I was a church plant intern for Acts2Fellowship, a campus-based christian fellowship at UCSB in it's second year after being founded. It was a tremendous experience and because the staff group was so small it operated a lot like a startup in that everyone had to do a little bit of everything. So while my main role was as a mentor to the UCSB students coming to our church, I was also responsible for cooking for the Friday meals (for a group of ~60 people) and running the audio systems for the Sunday services which amounted to running a full band with two main speakers, a subwoofer, floor monitors, 4 instruments, and a soundboard. Needless to say, I learned to learn quickly and picked up a lot of skills on the way.
